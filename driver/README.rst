f5-openstack-lbaasv1 Driver
===========================

The F5® OpenStack LBaaSv1 driver uses Neutron messaging queues to provision
load balancing tasks to the F5® LBaaSv1 agent(s). See the `project documentation <http://f5-openstack-lbaasv1.readthedocs.org/en/>`_ for installation and configuration instructions.
